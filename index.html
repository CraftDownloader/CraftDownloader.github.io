How to Install Minecraft Bedrock Minigames

Follow these steps to install and play any minigame downloaded from CraftDownloader.

  1. Download the .mcworld file from the minigame page.
  2. Open the downloaded file on your device.
  3. Minecraft Bedrock Edition will launch automatically and import the world.
  4. Go to Play → Worlds and locate the imported map.
  5. Start the world and follow any in-game instructions.
Note: These maps are designed for Minecraft Bedrock Edition. Make sure your Minecraft version is up to date.
