Prop Hunt

Description

Prop Hunt is a hide-and-seek style minigame where players disguise themselves as blocks while hunters search the map to find them before time runs out.

How to Play

Hiders transform into blocks and blend into the environment. Hunters must identify suspicious objects and eliminate all props before the timer expires.
